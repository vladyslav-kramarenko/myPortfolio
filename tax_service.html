Project information

  • Project on GitHub: taxService

EPAM Training project

This project imitates a website for tax reporting.

Supports 4 levels of users:

  • Unregistered user - has the ability to register
  • Registered user - has the ability to create and send reports. Supports loading and unloading in xml.
  • Inspector - has the ability to view submitted reports and accept or reject them. It has the ability to view statistics by users and types of reports.
  • Administrator - has the ability to view all reports, has the ability to change user rights.
All users have the ability to reset their password for their account through the email.

Technology in use:

  • Server: Apache Tomcat
  • Database: MySql
  • Backend: JSP, SQL, XML
  • Frontend: HTML, CSS, JavaScript, jQuery-ui, Bootstrap
